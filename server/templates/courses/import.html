Open Claude, attach your exam PDF, and paste the prompt below. Claude returns a CSV that imports cleanly here.

Open Claude ↗ 
I've attached an exam PDF. Extract every multiple-choice question into a CSV I can import into my study app.

Output the CSV as a downloadable .csv file. If you can't attach a file, output a single ```csv code block and nothing else.

Use exactly this header row, in this order:
Section,Category,Code,Question,A,B,C,D,Correct,Explanation,Source

Rules:
- One row per question.
- Section = the exam or paper name. Category = the topic the question belongs to.
- Code = a short unique id for each question (e.g. Q1, Q2, …). Never reuse a code.
- Question = the full question text. A, B, C, D = the four answer options (exactly four).
- Correct = the single letter A, B, C, or D of the right answer.
- Explanation = a short reason for the answer (optional). Source = page number or reference (optional).
- If a field contains a comma, wrap that whole field in double quotes.
- Only include questions actually in the PDF — never invent a question or an option.
- If the PDF doesn't state the correct answer, use your best judgement and note "answer inferred" in Explanation.
Then bring the CSV back here ↓
